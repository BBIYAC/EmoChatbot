## Django 실행 방법
**1. Repository 가져오기**
  ```
  git clone [REPO_URL]
   ```

**2. 가상환경 생성**
  ```
  python -m venv venv
  ```

**3. 가상환경 실행(window)** 
  ```
  source myvenv/Scripts/activate
  ```

**4. 패키지 설치**
  ```
  pip install -r requirements.txt
  ```

**5. makemigrations**
  ```
  python manage.py makemigrations
  ```

**6. migrate**
  ```
  python manage.py migrate
  ```

**7. 장고 서버 실행**
  ```
  python manage.py runserver
  ```
